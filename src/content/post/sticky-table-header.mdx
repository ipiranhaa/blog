## Solution

1. Wrap your table by a container then set max-width for fixed row header or set max-height for fixed vertical column of the table container
2. Make your fixed cells as sticky position, increase z-index layer, and set the point to sticky with the container

## Row header

```css
.container {
  max-height: 256px;
}

.header-cell {
  position: sticky;
  top: 0;
  z-index: 10;
}
```

## Bi-direction header

```css
.container {
  max-height: 256px;
  max-width: 768px;
}

.row-header-cell {
  position: sticky;
  top: 0;
  z-index: 10;
}

.column-header-cell {
  position: sticky;
  left: 0;
  z-index: 20;
}

.corner-header-cell {
  position: sticky;
  left: 0;
  top: 0;
  z-index: 30;
}
```
